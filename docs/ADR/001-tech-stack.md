# Tech Stack

| Layer | Technology | Reasoning |
|-------|------------|-----------|
| Language | Python | Quick MVP and deployment, libraries for web scraping and LLMs |
| Backend | FastAPI | Auto-generated API docs, async support |
| Web Scraping | BeautifulSoup + Playwright | Simple Python web scraping library with a fallback for more JS-heavy sites |
| LLM | Claude API | Classification for the most difficult examples |
| Database | PostgreSQL | Industry standard database for relational data |
| Scheduler | Cron | Simple job scheduler |
| Frontend | React + Tailwind | Simple framework for a frontend dashboard |
